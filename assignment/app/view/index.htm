1. http://users.sussex.ac.uk/~martinwh/3dapp/labs/ The main source of learning and reference for this website is as a basic template, used to quickly build the front-end pages of the website. The overall layout method is referenced, but most of the color matching, page layout, and content have been modified and redesigned. Some 3D Models JavaScript code has been referenced and referenced to achieve 3D Models camera perspective and lighting control, as well as motion rotation and other operations. But through imitation, a new rotation method and texture switching were achieved.
2. https://www.coca-cola.co.uk/ The brands images and models resources in the webpage, as well as the introduction text sources on the page, are generally stored in the app/data/data.json file and called through the front-end $. getJson method.
3. https://www.pexels.com/ Obtain and use background images from the front-end page and image resources from the gallery page on this website, all of which are open-source resources.
4. https://www.w3cschool.cn/ Learn basic HTML, CSS, and JavaScript operations.The basic framework and style modifications used to build web pages, as well as the modification and dynamic changes of page elements.
5. https://getbootstrap.com/ This website mainly uses the bootstrap front-end framework. This link is used to learn about the styles and components related to the framework, including webpage layout, tables, navigation, menu switching, and buttons.
6. https://fex.baidu.com/webuploader/ Image upload plugin, this plugin is used to help upload images through Ajax, using SPA pages to avoid refreshing the entire page. This plugin is referenced in the upload method on the gallery page, which is used in app/view/js/fileupload.js.
7. https://github.com/fancyapps/fancybox This plugin is used for zooming in on images, and these references are referenced in the home page and model page.
8. https://www.bilibili.com/video/BV1ZT411E7Gb/?spm_id_from=333.788.recommend_more_video.17 Learning and using Blender tools
9. https://fatfreeframework.com/3.8/home In order to meet the application structure of page MVC, this page uses the fatfreeframework to achieve this goal. Mainly drawing on the routing part and UI template engine of this framework, index.php implements page routing and jumping, the controller part implements parameter reception and page rendering of the front-end page, and the model implements modification and acquisition of database and JSON data.
10. https://www.cnblogs.com/ctb-web/p/9968288.html These code references are referenced in app/view/js/index.js.Because the website is a spa application, it will take a long time for the page to load for the first time. It needs to display pages loading to inform the user that the page resources are loading. Therefore, the syntax of ES6 needs to be used to implement this process, and the page has borrowed some of the ES6 code from this blog.
